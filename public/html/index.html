Mis Proyectos

The Chained Dungeon

Juego de aventuras inspirado en mazmorras. El objetivo es recorrer el mapa y eliminar a los enemigos que se interpongan en tu camino para poder escapar

Cuenta con distintos tipos de enemigos, items y niveles basados en XP

Desarrollado con:

  • C++
  • SFML

CodeFlix

Sitio web de una Biblioteca de películas.

Mediante el consumo de una API se obtienen las películas y se las divide por género.

Está conectada a una base de datos donde se recopila la información de cada película y se guardan los datos de los usuarios.

Cuenta con un sistema de início de sesión y de comentarios.

Desarrollado con:

  • HTML5
  • CSS3
  • JavaScript
  • MySQL
  • Node.JS

Burguer::Code

Sitio web de una Hamburgueseria.

Mayormente basada en el diseño y el impacto visual. Permite la navegación en el menú, la visualización de las sucursales y un formulario para postularte al trabajo

Desarrollado con:

  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap
